# Bounce
В игре вы управляете мячиком, и ваша цель - собрать все кольца, чтобы открыть портал и перейти на следующий уровень. В игре есть враждебные объекты, дополнительные жизни, изменение мячика с маленького на большой (у шарика меняются свойства), различные усиления мячика.
